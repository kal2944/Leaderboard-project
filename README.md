# Leaderboard-project

In this project I build an application called leader board which displays scores submitted by different players. It also allows you to submit your score.

## Built With

- Major languages
- HTML, CSS, Javascript
- Git
- Webpack

## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites
- Have LTS version of npm and node.js on your local environment.

### Setup
- Download or clone the repository. [repository page](https://github.com/kal2944/Leaderboard-project.git).

### Install
- Run npm install in the folder where your have the files.

### Usage
- Run npm start on your local path to start the web server.
